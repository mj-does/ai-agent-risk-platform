# AI Agent Risk Model & Platform

This project is a multi-agent system for security and risk analysis. It uses a graph-based approach to propagate risk from user prompts through agents and tools to critical target systems.

## Project Structure
- `backend/`: FastAPI backend services.
- `agents/`: AI agents for various tasks.
- `graph/`: Graph database schema (TigerGraph) and GSQL queries.
- `frontend/`: React-based user interface.
- `docs/`: Project documentation.
